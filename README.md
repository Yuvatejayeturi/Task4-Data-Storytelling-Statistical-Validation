# Netflix Content Analysis Dashboard

## Project Overview
This project analyzes Netflix content data using Python and Power BI to identify trends, patterns, and business insights.

## Objectives
- Analyze Netflix content distribution
- Explore country-wise content production
- Analyze ratings and release trends
- Validate findings using statistical methods

## Tools Used
- Python
- Pandas
- NumPy
- SciPy
- Power BI
- Microsoft Word
- Microsoft PowerPoint

## Dataset Summary
- Total Records: 8809
- Movies: 6131
- TV Shows: 2676

## Dashboard Insights
- Movies dominate Netflix content.
- United States contributes the highest number of titles.
- India ranks second in content production.
- TV-MA is the most common rating.
- Content production peaked around 2019.

## Statistical Validation
Hypothesis testing was performed using the Chi-Square test.

Chi-square statistic:
1355.40

P-value:
1.03 × 10^-296

Result:
The null hypothesis was rejected, confirming a significant difference between Movie and TV Show distributions.
